# CSE108 Student Enrollment Web App

This is a Flask-based student enrollment system with login functionality and role-based dashboards for **students**, **instructors**, and **admins**. Admins can manage users (create, edit, delete) and see all registered users in the database.

---

## Tech Stack

- Python 3.x
- Flask
- Flask-Login
- SQLite (via SQLAlchemy)
- HTML + Jinja templates
- CSS (for basic styling)

---

## Getting Started

These instructions will help you run the project locally using either **VS Code** or **PyCharm**.

---

### 1. Clone the Repository

```bash
git clone https://github.com/ZEnglish89/CSE108-08.git
cd CSE108-08
```
---

### 2. (Optional) Create a Virtual Environment

For MacOS/Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```

For Windows
```bash
python -m venv venv
venv\Scripts\activate
```
---

### 3. Install required packages

```bash
pip install -r requirements.txt
```
---

### 4. Run the app

```bash
python app.py
```

Then open your browser and go to:
```
http://127.0.0.1:5000/ 
```
